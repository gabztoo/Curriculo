Mass Email Sending using Tkinter and Pandas

Envio de E-mails em Massa utilizando Tkinter e Pandas

This is an .exe program that follows a Python script using the Tkinter library to create a simple graphical interface and allow mass email sending. The script loads data from an Excel file, fills in the email fields with personalized information, and allows attachments before sending the emails.

Este é um Programa .exe que segue script Python que utiliza a biblioteca Tkinter para criar uma interface gráfica simples e permitir o envio de e-mails em massa. O script carrega dados de um arquivo Excel, preenche os campos do e-mail com informações personalizadas e permite anexar arquivos antes de enviar os e-mails.

Features:

Funcionalidades:

Excel File Selection: Allows selecting the Excel file containing the recipients and other necessary information.

Seleção de Arquivo Excel: Permite selecionar o arquivo Excel contendo os destinatários e outras informações necessárias.

Sender Authentication: Requests the sender's email and password for authentication on the SMTP server. It is recommended to use a Google app password for greater security.

Autenticação do Remetente: Solicita o e-mail e a senha do remetente para autenticação no servidor SMTP. Recomenda-se usar uma senha de aplicativo do Google para maior segurança.

Email Field Personalization: Uses the information from the Excel file to fill in the email fields (e.g., name, email address).

Personalização dos Campos do E-mail: Utiliza as informações do arquivo Excel para preencher os campos do e-mail (ex: nome, endereço de e-mail).

Attachments: Allows attaching files to the email before sending.

Anexos: Permite anexar arquivos ao e-mail antes de enviar.
